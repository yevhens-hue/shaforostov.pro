# Developer Portfolio Showcase

A lightweight, purely front-end interactive resume and portfolio. Built for maximum speed and SEO optimization to cleanly display past projects, tech stack experience, and professional background.

## Architecture Highlights
- **Zero Dependencies**: Vanilla HTML/CSS architecture for instant load times (perfect Lighthouse scores).
- **Responsive Design**: Fluid UI that adapts seamlessly to desktop, tablet, and mobile browsers.
- **Tech Stack**: HTML5, CSS3.
